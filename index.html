Create Tooltips from Alt Text

This is a simple JQuery function to select every tag with the class ".altText" and create a bootstrap tooltip. This puts visually impaired readers first while automating the tooltip process for the developer. Don't forget alt text!

$(document).ready(function(){
    $(".altTitle").each(function (index) {
        var thisAltText = $(this).attr('alt');
        $(this).attr('title', thisAltText);
        $(this).attr('data-toggle', 'tooltip');
        });

    $('[data-toggle="tooltip"]').tooltip(); //enable bootstrap tooltips
    });
